# Narrative Context Graph

A narrative analysis and story scaffolding tool that maps fiction across 11 narrative dimensions. Build story arcs from genre templates, blend genres, get beat-level writing suggestions, and reverse-engineer existing manuscripts with AI-powered analysis.

## Quick Start

### Option 1: Open the built version directly

Open `dist/index.html` in any modern browser. No server needed.

### Option 2: Run the dev server

Requires [Node.js](https://nodejs.org/) (v18+).

```
npm install
npm run dev
```

Or use the start scripts:

- **Mac/Linux:** `./start.sh`
- **Windows:** `start.bat`

### Option 3: Preview the production build

```
npm run build
npm run preview
```

## AI Features

The AI chat panel and chapter analysis require an [OpenRouter](https://openrouter.ai/) API key. Enter your key in the Analyze workflow or use the chat panel. Your key is stored locally in your browser and sent only to the OpenRouter API.

## Workflows

- **Scaffold** -- Build story arcs from genre templates with 11-dimension beat editing, genre blending, and AI-powered suggestions
- **Analyze** -- Paste chapter text and get AI-scored dimensional analysis, gap detection, and editorial recommendations
- **Chat** -- Click the left-edge toggle to open the AI assistant, which can read and modify all fields in the active workflow
